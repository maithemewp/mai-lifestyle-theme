MAI LIFESTYLE THEME
https://maitheme.com

INSTALL
1. Upload the Mai Lifestyle Theme folder via the WordPress dashboard, Appearances > Themes > Add New or manually via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Mai Lifestyle Theme via Appearances > Themes in your WordPress dashboard.
4. Customize your site via your wordpress dashboard Appearances > Custmomize and configure to your liking.
